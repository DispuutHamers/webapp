    Introduction

    Welcome to the Hamers APIv2, you can use this API to access and update the various resources that are available on our website.

    This API exclusivly uses JSON in its requests so you are advised to set the content type of your requests to application/json

    This is an updated version of the current API documentation and is still a work in progress.

    Authentication

    Authorization: Token token="abcdefghijklmopqrstuvwxyz"

    This resource has no output

    Make sure to replace abcdefghijklmnopqrstuvwxyz with your API key.

    Hamers uses API keys to allow access to the API. You can register a new API key at your account page.

    Hamers expects for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: Token token="abcdefghijklmopqrstuvwxyz"

    You must replace abcdefghijklmnopqrstuvwxyz with your personal API key.

    Quotes

    Get All Quotes

    id, text, user_id, created_at

    [
  {
    "id": 1,
    "text": "Dit is sample data uit een quote",
    "user_id": 1,
    "created_at": "2014-03-18T15:12:31.000+01:00"
  },
  {
    "id": 2,
    "text": "Dit zou zomaar een andere quote zijn",
    "user_id": 2,
    "created_at": "2014-03-18T15:12:47.000+01:00"
  }
]

    This endpoint retrieves all quotes.

    HTTP Request

    GET https://zondersikkel.nl/api/v2/quotes

    Query Parameters

    Parameter Default Description
    sorting nil If set to date-asc will sort by created_at ascending if set to date-desc will sort by created_at descending.

    "user_id": 2,
    "created_at": "2014-03-18T15:12:47.000+01:00"
  }

    This endpoint retrieves a specific quote.

    HTTP Request

    GET https://zondersikkel.nl/api/v2/quote/<ID>

    URL Parameters

    Parameter Description
    ID The ID of the quote to receive.

    Update a Quote

    text, user_id

    The JSON here is a valid body

      {
    "text": "Dit zou zomaar een andere quote zijn",
    "user_id": 2
  }

    This endpoint updates a quote.

    HTTP Request

    UPDATE https://zondersikkel.nl/api/v2/quote/<ID>

    URL Parameters

    Parameter Description
    ID The ID of the quote to update.

    Request body

    Parameter Required Description
    text No The text of the quote
    user_id No The person that said the text
    This returns the `JSON` of the updated quote.

    Post a new Quote

    text, user_id

    The JSON here is a valid body

      {
    "text": "Dit zou zomaar een andere quote zijn",
    "user_id": 2
  }

    This endpoint created a new quote.

    HTTP Request

    POST https://zondersikkel.nl/api/v2/quote

    Request body

    Parameter Required Description
    text Yes The text of the quote
    user_id Yes The person that said the text
    This returns the `JSON` of the new quote.

    id, name, soort, created_at, picture, percentage, brewer, country, URL, cijfer

    [
  {

    "name": "Jopen Laphroaig Barrel Aged.",

    "picture": "http://hereisapic.nl/img.png",
    "percentage": "10.0%",
    "brewer": "Jopen",
    "country": "Nederland",
    "URL": "",
    "cijfer": 8.75
  },
  {
    "id": 2,
    "name": "Snake Dog IPA",
    "soort": "Indian Pale Ale",
    "created_at": "2014-04-22T00:48:49.000+02:00",
    "picture": "http://hereisapic.nl/img.png",
    "percentage": "7.1%",
    "brewer": "Flying Dog",
    "country": "VS",
    "URL": null,
    "cijfer": 7.13
  }
]

    This endpoint retrieves all beers.

    HTTP Request

    GET https://zondersikkel.nl/api/v2/beers

    Query Parameters

    Parameter Default Description
    reviewed nil If set to true will only show beer that has been reviewed by the owner of the API key, if set to false will only show beers that have not been reviewed by the owner of the API key.

    Fields

    Field Description
    id The unique identifier of this entry
    name The name of the beer
    soort The type of the beer
    created_at The date this beer was added to our database
    picture A URL containing a picture of this beer
    percentage The alocohol content of this beer
    brewer The brewer of this beer
    country The country this beer originates from
    URL A URL to more information about this beer
    cijfer The avarage grade our members give this beer
    Not all fields need to be populated for every beer.

    Push Notifications

    FCM

    device

    {
  "device":"deviceidhere"
}

    At this endpoint you can register a device to receive push notification via FCM

    HTTP Request

    POST https://zondersikkel.nl/api/v2/register

    Request body

    Parameter Required Description
    device Yes The device ID to register
    It is currently not possible to unregister, to do this send a fake `device_id`

    Errors

    The Hamers API uses the following error codes:

    Error Code Meaning
    400 Bad Request – Your request sucks
    401 Unauthorized – Your API key is wrong
    404 Not Found – The specified resource could not be found
    406 Not Acceptable – You requested a format that isn’t json
    418 I’m a teapot
    429 Too Many Requests – You’re requesting too many kittens! Slow down!
    500 Internal Server Error – We had a problem with our server. Please contact the admin.
